
## Installation
    $ git clone https://github.com/formanite-20/formanite-20.github.io.git
    $ cd formanite-20.github.io.git
    $ npm install

## Start dev server
    $ npm start
    Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Simple build for production
    $ npm run build
